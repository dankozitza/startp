This program, getconc, will generate a concordance for the file entered on the
command line. It will work on any text file.

The following are commands to be used for building the executable 'getconc'.

Compile:

   g++ -c concordance.cpp -c getconc.cpp

Link:

   g++ concordance.o getconc.o -o getconc

Execute:

   ./getconc FILE

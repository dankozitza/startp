modsets is a menu-driven program for modifying sets of items.

Commands:
   h          - print this help message
   p          - print sets that have items in them
   i v n      - insert item v into set n
   r v n      - remove item v from set n
   + n1 n2 n3 - assign the union of set n1 and set n2 to set n3
   - n1 n2 n3 - assign the intersection of set n1 and set n2 to set n3
   m n1 n2    - copy set n1 to set n2
   s n        - print the size of set n
   f v n      - determine if item v is found in set n
   w n        - write set n to stdout
   c n        - clear set n
   q          - exit this program
